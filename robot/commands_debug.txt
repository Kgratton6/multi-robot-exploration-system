Commands :

Run no machine : /usr/NX/bin/nxplayer

create a new ros package (cd src/limo_ros2) : 
1. cd src/limo_ros2
2. ros2 pkg create ros_gz_modules --build-type ament_python --dependencies rclpy

add source humble to bash : echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc

Get permission from ttyTHS1 : sudo chmod 666 /dev/ttyTHS1
Get permission from ttyUSB0 : sudo chmod 666 /dev/ttyUSB0
